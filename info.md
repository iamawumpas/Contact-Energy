# Contact Energy — Home Assistant Custom Integration

**Version:** 0.0.1

![Contact Energy Logo](https://raw.githubusercontent.com/iamawumpas/contact-energy/main/logo.svg)

A streamlined Home Assistant integration for Contact Energy (New Zealand) that provides account information and energy usage data for the Energy Dashboard and ApexCharts.

- **Domain:** `contact_energy`
- **Installation:** HACS or manual copy to `custom_components/contact_energy`
- **Status:** Stable release

## Features

- Energy consumption tracking with Home Assistant statistics integration
- Account balance and billing information sensors
- Free electricity usage tracking (for special plans)
- Energy Dashboard integration
- ApexCharts compatibility for historical data visualization
- Smart data download (only fetches missing days after initial setup)
- Memory efficient design

## Requirements

- Home Assistant 2023.1.0 or later
- Contact Energy account (New Zealand)
- Internet connection for API access

For setup and usage instructions, refer to the project README.
